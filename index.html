Актуальность

Беспилотные летательные аппараты (БПЛА) как никогда актуальны. Так как из-за протяженности железных дорог, установить наблюдение за каждым участком очень проблематично и затратно. Для выявлений несанкционированных проникновений, распознаваний ЧС, а также при прокладывании железнодорожных путей, удобнее использовать мобильны дроны.

Техническая часть

PointFusion - сеть, предназначенная для слияния, которая использует «PointNet» вместе с детектором 2D-объектов. Общая сетевая архитектура показана на рисунке. Как видно на рисунке, есть 2 ветви - одна для облака точек и одна для изображений. Ветка для изображений передает изображение в нейронную сеть «ResNet» для извлечения признаков по которым определяется вторжение на объект. Для ветви облака точек выбираются только точки, которые можно проецировать на двумерное изображение. Функции извлекаются в этой ветке с помощью нейронной сети «PointNet».

Функция изображения, глобальная функция облака точек и функция облака точек для каждой точки создают изображение объединенного объекта. Затем объединённое изображение проходит обработку с помощью 3D угла смещения, в которой определяются размеры в пространстве, объекта вторгшегося на инфраструктуру. После выводит изображение с четкими границами постороннего объекта. В данной подзадаче мы изучаем программное обеспечение сети PointFusion. Она состоит из двух нейросетей. Язык программирования нейросетей – Python. «ResNet» - нейросеть, предназначенная для выделения конкретного объекта на 2D изображении. «PointNet» - нейросеть, предназначенная для выделения облака точек, полученных с LIDAR.

Как мы написали ранее обработка с помощью 3D угла смещения имеет 2 вывода. Первый вывод – берем абсолютное значение 8 углов ограничивающей рамки 3D. Для этого мы будем использовать функцию «vert» и библиотеки «Math и Matrix» на языке программирования Python. Функцию «vert» будем задавать с помощью 3-х координат (x, y, z) для каждой вершины параллелепипеда. Координаты для функции будем брать из полученного объединённого изображения с камеры высокого разрешения и LIDAR. Итого мы получим 8 координат, которые будут заключать объект, который вторгся на инфраструктуру, в 3D рамки.

Второй вывод - относительное смещение и оценка от каждой точки облака точек. Для этого мы будем использовать матрицы. В Python мы задаем на нужной нам области точек матрицу, которая будет определять положение, масштаб, вращение 3D рамки в пространстве, для заключения нужного объекта в рамку.

Для перемещения в пространстве будем использовать метод Translation. Мы будем менять значения положения вдоль осей координат X, Y, Z. Для этого нам понадобится исходная матрица, полученная с нужной нам области и функция Matrix, в которой мы и задаем значения положения вдоль осей координат.

Для масштабирования мы будем использовать функцию scale.factor. Она состоит из трех аргументов. Первый – коэффициент масштабирования (он отвечает во сколько раз нужно увеличить 3D рамку). Второй – размер матрицы, поскольку мы работаем с 3D изображением, он всегда будет равен матрице 4x4. Третий – определяет по какой оси будет масштабирование. Задавать будем координатами (x, y, z), например (1, 0, 0) – масштабирование будет происходить только по оси X.

Далее собираем все вместе и получаем нужную нам 3D рамку, заданную в пространстве. Затем накладываем полученную рамку на изображение, полученное с камеры высокого разрешения и LIDAR. В итоге мы получаем конечное изображением с определением объекта вторгшегося на инфраструктуру.

Охрана местоположения дронов от злоумышленников

Первым шагом в эффективной защите от угрозы является понимание того, что она существует. Важным аспектом эффективной охраны периметра является возможность вести круглосуточное наблюдение, независимо от погодных условий, условий окружающей среды или времени суток. Тепловизионные камеры могут обеспечить постоянную защиту от потенциальных злоумышленников, независимо от визуальных ограничений, уровня освещенности и других факторов.

Периметр аэропорта требуют не только непрерывного видеонаблюдения и контроля, но и надежного обнаружения вторжения с минимально возможным количеством ложных сигналов тревоги. Именно поэтому расширенная видео аналитика становится обязательной для любой адекватной системы видеонаблюдения. Такие функции, как обнаружение событий в реальном времени и оптимальное предоставление данных, оказывают существенную помощь в анализе попыток вторжения и содействуют упреждающему реагированию.

Расширенная аналитика, как правило, обеспечивает несколько автоматических вариантов уведомления о тревоге. Высококачественная, уникальная и адаптированная аналитика способна исключить ложные сигналы тревоги, понизить совокупную стоимость владения системой для конечного заказчика, позволяя при этом персоналу службы безопасности эффективно реагировать на события. Конечно все перечисленные выше технические средства и технологии наблюдения за периметром аэропорта, такие как радары, тепловизионные и обычные камеры, работают сами по себе достаточно продуктивно, но использование всех этих технологий и подсистем вместе обеспечивает наличие эффективной системы безопасности периметра. Например, радиолокатор наземного наблюдения может обнаруживать возможные угрозы за пределами линии ограждения по мере приближения какого-либо объекта и посылать сигнал PTZ-камерам для позиционирования их при необходимости наблюдения за определенной зоной. Встроенная видео аналитика на борту камер позволяет идентифицировать объект, сообщить о нарушении персоналу службы безопасности и собрать дополнительные данные, например, с помощью функции распознавания лиц и использования LIDAR совместно с камерами.
